# Drishti

## AI-Powered Crime Intelligence & Investigation Assistant

### Overview

Drishti is an AI-powered conversational crime intelligence platform designed to help investigators, analysts, and policymakers interact with crime records using natural language.

Instead of manually searching through thousands of records, users can simply ask questions in everyday language. Drishti retrieves relevant information, analyzes crime patterns, identifies relationships between people, locations, and incidents, and provides evidence-backed insights to support faster and smarter investigations.
